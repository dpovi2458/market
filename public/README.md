# Market Facultad - Public Assets

## Archivos Incluidos

- `favicon.svg` - Ícono del navegador (carrito SVG)
- `manifest.json` - Configuración PWA
- `robots.txt` - Instrucciones para buscadores
- `feed.xml` - RSS feed
- `logo.png` - Logo de la marca
- `market facultad.png` - Versión extendida del logo

## Para Completar Manualmente

Si deseas optimizar aún más, puedes agregar:

1. **apple-touch-icon.png** (180x180px)
   - Ícono para favoritos iOS

2. **og-image.png** (1200x630px)
   - Imagen para compartir en redes sociales

3. **icon-192.png** y **icon-512.png**
   - Ícono de la app en Android
   - Generar con herramientas online o Figma

## Comandos Útiles

```bash
# Ver cómo se ve en desarrollo
npm run dev

# Compilar y verificar
npm run build
```
